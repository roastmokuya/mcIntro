## Introduction

使用 React 18 + Vite 5 + TypeScript + Three.js + Tailwindcss 3 實作 3D 自我介紹。

本網站僅作為個人作品練習用，並無任何營利行為，3D 素材來自 sketchfab，音樂來自 Suno。

## Project Description

- npm install
- npm run dev
- Node.js v18.20.2
